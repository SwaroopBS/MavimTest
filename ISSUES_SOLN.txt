

To solve the long path of >150 char, we need to download the latest version of visual studio with C++


Installing dlib:


1st install the python v3.8.9 using : https://www.python.org/downloads/release/python-389/

To check the Environment Var. goto : Control Panel\System and Security\Security

Need to move the version py v3.8.9 up in the Environment var. list

https://www.youtube.com/watch?v=AUJKdehF2ZA

https://github.com/datamagic2020/Install-dlib

Install dlib after unzipping C:\Users\user\Downloads\Install-dlib-main\Install-dlib-main\.... 


Installing cmake:

pip install cmake


Installing Face Recognition module:

pip install face_recognition


To see the pip modules installed:

C:\Users\user\AppData\Local\Programs\Python\Python38\Scripts
